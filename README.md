## University DataBase SQL Project

Sistema de gestión de mascotas desarrollado con **Java + Maven**, interfaz gráfica en **Java Swing** y base de datos MySQL usando **XAMPP**.

##

- Java
- Maven
- Java Swing 
- MySQL
- XAMPP (Apache + MySQL)

---

##  Descripción

**PetSystem** es una aplicación de escritorio con interfaz gráfica que permite gestionar información de mascotas.

Incluye funcionalidades como:

- Registro de mascotas 
- Selección de datos mediante ComboBox 
- Conexión a base de datos SQL 
- Visualización de información en la interfaz 

---

##  Requisitos

- usar xampp y abrir pet system en un IDE
